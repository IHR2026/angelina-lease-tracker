# Angelina County Oil & Gas Lease Tracker

Interactive web map tracking oil and gas leases, parcel ownership, and well locations in Angelina County, Texas.

## Features

- **709 deduplicated oil & gas leases** color-coded by operator (Aethon, BP America, Chesapeake, EOG, etc.)
- **60,763 county parcel boundaries** with owner names, acreage, legal descriptions, and market values — sourced from the Angelina County Appraisal District
- **1,013 abstract boundaries** with survey names
- **101 horizontal wells** showing surface-to-bottomhole lateral paths, color-coded by status (Active, Shut-In, DUC, etc.)
- Leased parcels are **color-filled** by operator, matching lease abstracts to parcel boundaries
- Click any feature for detailed popup info
- Search and filter leases by operator, grantor, date range, or text
- Export filtered results to CSV

## Quick Start

### 1. Download parcel data (one-time, ~2 min)

The parcel boundary file is too large for GitHub (~43 MB). Run the download script to pull it from the Angelina CAD ArcGIS service:

**Windows (Python):**
```bash
python download_data.py
```

**Windows (PowerShell, no Python needed):**
```powershell
powershell -ExecutionPolicy Bypass -File download_data.ps1
```

### 2. Start a local web server

**Python:**
```bash
python -m http.server 8080
```

**Node.js:**
```bash
npx serve .
```

### 3. Open in browser

Navigate to [http://localhost:8080/angelina_lease_tracker.html](http://localhost:8080/angelina_lease_tracker.html)

## Data Sources

| Layer | Source | Records |
|-------|--------|---------|
| Leases | DrillingInfo / Enverus export | 709 (deduplicated from 14,865) |
| Parcels | Angelina CAD ArcGIS FeatureServer | 60,763 |
| Abstracts | Angelina CAD ArcGIS FeatureServer | 1,013 |
| Wells | DrillingInfo / Enverus export | 101 |

## Files

- `angelina_lease_tracker.html` — Main web application
- `leases_data.json` — Deduplicated lease data (GeoJSON)
- `angelina_abstracts.geojson` — Abstract boundaries
- `angelina_wells.geojson` — Well laterals and data
- `download_data.py` — Python script to download parcel data
- `download_data.ps1` — PowerShell script to download parcel data
- `angelina_parcels.geojson` — *(generated by download script)*
